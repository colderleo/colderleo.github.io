mysql

mysql location:

/usr/local/mysql

查看mysql服务是否启动：

service mysqld status 或者 service mysql status

启动或者或者重启mysql服务：

service mysql start ；

service mysql restart；

service mysql stop

mysql命令行登陆root：

mysql -uroot -p

然后会提示输入密码
