volatile

https://www.runoob.com/w3cnote/c-volatile-keyword.html

# 作用

  • 单线程：防止编译器优化
  • 多线程：防止从寄存器中读取变量的值

# 指针声明

2、volatile 指针和 const 指针类似

  • 常量指针 const int* p 常量的指针
  • 指针常量 int* const p 指针本身是常量

修饰由指针指向的对象、数据是 const 或 volatile 的：

const char* cpch;
volatile char* vpch;
1
2

指针自身的值——一个代表地址的整数变量，是 const 或 volatile 的：

char* const pchc;
char* volatile pchv;
