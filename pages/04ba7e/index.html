微擎安装使用

# 安装

注意事项

微擎1.7.0安装时是自己手动创建数据库，然后运行.sql文件，再修改配置数据库文件对接

1.7.5之后的版本将微擎复制过去后，直接进入站点目录，url中可能需要加index.php，然后会打开一个网页在线安装。不需要手动创建数据库，之类的。但是输入数据库名字的时候，不能带点，比如w7.1.7.5，就不行，会出错。可命名为w7175

# 微擎数据库更新

# 网上的参考

https://v.qq.com/x/page/h0712gycn4p.html

数据库对比前，必须备份好，备份好，备份好，出错了还有得救，在运营的，建议晚上操作

数据库对比

找出模块后台文件的manifest.xml文件，一个现有的版本xml，一个新版本的xml

下载对比文件Beyond Compare 4 下载地址http://www.beyondcompare.cc/xiazai.html

准备一个SQL文件，注意格式，SQL

打开Beyond Compare 4点击文本对比，把新旧版本的xml文件，拉进软件里面做对比

这里的查看，红色的就是标注增加的数据语句，注意数据表只有多没有少了的。

把多出来的，语句在哪个数据表，一一复制出来，看视频演示

alter table ims_zhtc_account add money decimal(10,2) NOT NULL after cityname;

黄色为数据表名，红色为语句，复制的时候注意标点符号不能少，不能多，后面是结束语句

对比好数据，直接导入微擎数据库，

把以前版本的文件直接改个名字，上传新版后台文件，更新后台缓存，测试看是否有问题，没问题完成更新
